O grupo de pesquisa

Este é um projeto do grupo de pesquisa Formação docente e tecnologias de informação e comunicação (veja lista completa de participantes no CNPq) que desde 2011 estuda a Universidade Aberta do Brasil (UAB). O grupo é coordenado pela UFF com parceria da UEL, UNICAMP e UnB. Os resultados aqui apresentados são o resultado de um estudo inicado em 2013 com enfoque nos Núcleos de Educação a Distância da Universidade Aberta do Brasil, conduzido em parceria com a Cátedra UNESCO em Educação Aberta na UNICAMP.

Este é um grupo de pesquisa independente e não representa a posição ou opiniões da CAPES ou do Sistema Universidade Aberta do Brasil.
O projeto

O mapa apresenta duas bases de dados:

Primeiro, uma base de informações extraídas do SisUAB em março de 2016, tornando público o acesso a informações sobre a UAB. Trazemos a tona um mapa com todas as IPES (Instutições Públicas de Ensino Superior) e todos os polos ativos, que juntos, compõe o Sistema UAB. Para cada um desses, apontamos um histórico detalhado sobre cursos e alunos participantes. Vale ressaltar que não incluímos os cursos que não tiveram oferta real de início, ou seja, cursos que podem ter sido planejados mas não chegaram a ser efetivados. Os dados de geolocalização são aproximados. Buscamos apontar com a maior especificidade possível, a localização das IPES (feitas manualmente). Os polos forma geolocalizados automaticamente e referem a cidade, sem precisão no nível de logradouro.

Segundo informações detalhadas sobre os Núcleos de Educação a Distância de 8 universidades que participaram de uma pesquisa realizada pelo nosso grupo focada na gestão de recursos educacionais no âmbito da UAB. Para essas instituições incluímos um texto introdutório e uma detalhada linha do tempo de sua participação na UAB. Esses espaços são responsáveis pela articulação, dentro da universidade, para a efetivação dos cursos oferecidos pela UAB. Como cada universidade tem autonomia para definir seu modelo de ação, cada um desses núcleos se estrutura de maneira diferente. O objetivo desses estudo é descrever e avaliar questões relativas ao funcionamento e estrutura desses centros, com particular atenção aos processos de produção e disseminação de recursos educacionais.

Fizemos o possível para garantir que os dados estão corretos, dadas as limitações apontadas acima. Caso identifique algum problema, ou queira ajudar a melhorar o sistema, entre em contato colocando um issue na nossa página no Github.

O software

O sistema que desenvolvemos, apelidado de Ciclope, é um software livre, que agrega a funcionalidade de diversas ferramentas Existem duas versões do sistema a primeira não depende de banco de dados e faz uso inteiramente de dados disponíveis no formato JSON, fazendo uso de HTML + Javascript (Ciclope). A segunda, que utilizamos nesse site, foi customizada para fazer uso de um banco de dados SQL, usa Javascript + HTML + PHP (CiclopeDB). Ambas estão licenciadas com GPL v3.0.
