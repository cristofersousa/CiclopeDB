O grupo de pesquisa

Este é um projeto do grupo de pesquisa Formação docente e tecnologias de informação e comunicação (CNPq) que desde 2011 estuda a Universidade Aberta do Brasil (UAB). Os resultados aqui apresentados são de um estudo inicado em 2013 com enfoque nos Núcleos de Educação a Distância da Universidade Aberta do Brasil. O estudo foi conduzido em parceria com a Cátedra UNESCO em Educação Aberta na UNICAMP. Este é um grupo de pesquisa independente e não representa a CAPES ou a Universidade Aberta do Brasil.
Software

O sistema, apelidado de Ciclope, é um software livre, que agrega a funcionalidade de diversas ferramnetas. Existem duas versões do sistema, a primeira não depende de banco de dados e faz uso inteiramente de dados disponíveis no formato JSON (Ciclope). A segunda, que utilizamos nesse site, foi customizada para fazer uso de um banco de dados SQL (CiclopeDB).
